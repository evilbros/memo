# install nodejs

* centos

```
curl -sL https://rpm.nodesource.com/setup_lts.x | bash -
yum install -y nodejs
```

* ubuntu

```
curl -sL https://deb.nodesource.com/setup_lts.x | bash -
apt install -y nodejs
```
